# 📚 Java 语言基础
## 一、什么是面向对象？ 🎯
> * **面向对象**(OOP):是将拥有共同属性和方法的一类事物封装为"类"(class),通过实例化"对象"来管理数据,调用方法,通过对象的交互来实现程序的功能的一种编程范式


## 二、静态方法和构造方法有什么区别？ ⚖️
> * **静态方法**:static修饰,属于类,由类名调用,只能访问静态成员而不能访问实例成员,应用于工具类方法和main方法
> * **构造方法**:名称与类名相同,无返回类型,不能直接调用,new时由系统自动调用,可以访问静态和实例成员,应用于创建对象时对成员变量进行***赋值***(***不是用来创建对象的***)

## 三、Java 修饰符 🔧
### 3.1 Java 中的修饰符及使用场景
#### 📌 访问控制修饰符:控制成员可被访问的范围
> * **public**: ️所有类 1️⃣当该"类"需要被其他类访问 2️⃣当该"方法"需要被外部调用
> * **protected**: 本类,本包,不同包的子类
> * **缺省**: 本类,本包中访问
> * **private**: 仅限于本类访问️1️⃣类的成员变量 2️⃣该类自己的辅助方法 
#### 📌 非访问控制修饰符
> * **static**: 属于类,可由类名直接调用  
> * **final**: "类" -> 不可被继承,"方法" -> 不可被重写,"变量" -> 不可被修改
> * **abstract**: 
  

### 3.2 各修饰符的作用对象？
📌
> * **public**:成员变量, 方法, 构造方法, 类
> * **protected**:成员变量, 方法, 构造方法(不能修饰类)
> * **缺省**:类, 成员变量, 方法, 构造方法, 类
> * **private**:成员变量, 方法, 构造方法(不能修饰类) 

📌
> * **static**:变量, 方法, 代码快, 内部类 (不能修饰普通类)
> * **final**:类, 方法, 变量
> * **abstract**:类, 方法
