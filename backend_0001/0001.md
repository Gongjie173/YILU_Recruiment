# 📚 Java 语言基础
## 一、什么是面向对象？ 🎯
> * **面向对象**(OOP):是将拥有共同属性和方法的一类事物封装为"类"(class),通过实例化"对象"来管理数据,调用方法,通过对象的交互来实现程序的功能的一种编程范式
> * **封装**:将数据和方法绑定在一起,隐藏细节,提供接口
> * **继承**:子类可以继承父类的某些属性和方法,减少代码冗余
> * **多态**:有继承/实现关系,父类引用指向子类对象

## 二、静态方法和构造方法有什么区别？ ⚖️
> * 静态方法:static修饰,属于类,由类名调用,只能访问静态成员而不能访问实例成员,应用于工具类方法和main方法
> * 构造方法:名称与类名相同,无返回类型,不能直接调用,new时由系统自动调用,可以访问静态和实例成员应用于创建对象时进行初始化

### 3.1 Java 中的修饰符及使用场景
#### 📌 访问控制修饰符:控制成员可被访问的范围
> * public: ️所有类均可访问 1️⃣当该"类"需要被其他类访问 2️⃣当该"方法"需要被外部调用
> * protected: 本类,本包各类,不同包的子类
> * 默认(缺省): 在本类,本包中访问
> * private: 仅限于本类访问️1️⃣类的成员变量 2️⃣该类自己的辅助方法
